# Auth Service Frontend

Простой React-фронтенд для ручного теста `auth-service`.

## Что умеет

- `register`
- `login`
- `refresh`
- открыть Google OAuth flow

## Запуск

```bash
cd auth-service-frontend
npm install
npm run dev
```

По умолчанию UI ходит в `http://localhost:8080`, но адрес можно поменять прямо в интерфейсе.
